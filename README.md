# SumGame

SumGame 是一個基於 Python 和 Tkinter 的桌面遊戲，靈感來自手機遊戲 [Number Sums](https://play.google.com/store/search?q=number%20sums&c=apps&hl=zh_TW)。玩家需要在一個網格中選擇正確的格子，以匹配每行和每列的目標總和。遊戲結合了邏輯推理和數學計算，提供挑戰性和娛樂性。

## 遊戲特色

- **邏輯推理**：玩家需要根據行和列的總和來推斷哪些格子是正確的答案。
- **隨機生成**：每次遊戲都會隨機生成新的網格和答案，提供無限重玩性。
- **提示功能**：可以使用提示來自動選中一個正確的格子，但會扣除分數。
- **右鍵標記**：可以右鍵點擊來標記或清除格子，避免誤選。
- **分數系統**：正確選中答案格子加1分，使用提示扣1分。
- **勝利條件**：選中所有正確的答案格子即可獲勝。

## 如何玩

1. **啟動遊戲**：運行 `Start.py` 來啟動遊戲設定視窗。
2. **設定參數**：
   - 網格大小 (x)：橫向格子數量。
   - 網格大小 (y)：縱向格子數量。
   - 答案格比例 (%)：正確答案格子佔總格子的百分比（1-100）。
3. **遊戲規則**：
   - 每個格子顯示一個數字（1-10）。
   - 左邊和上邊的標籤顯示該行/列所有答案格子的總和（目標總和）。
   - 右邊和下邊的標籤顯示已選中答案格子的總和和進度。
   - 左鍵點擊：選中格子。如果是正確答案，加分；如果是錯誤，遊戲結束。
   - 右鍵點擊：標記或清除格子。如果清除正確答案，遊戲結束。
   - 使用提示按鈕來自動選中一個未選中的正確格子（扣1分）。
4. **勝利**：選中所有正確答案格子。
5. **失敗**：選中錯誤格子或清除正確答案格子。
6. **重新開始**：點擊「重新開始」按鈕回到設定視窗。

## 安裝和運行

### 系統需求
- Python 3.6 或以上版本
- Tkinter（通常隨 Python 安裝）

### 遊玩步驟
1. clone或下載此專案到本地。
2. 確保 Python 已安裝。
3. 運行遊戲：
   ```bash
   python Start.py
   ```

### 檔案結構
- `Start.py`：遊戲啟動和設定視窗。
- `model.py`：遊戲數據和核心邏輯（Model）。
- `view.py`：Tkinter 介面元件（View）。
- `controller.py`：協調 Model 和 View（Controller）。

## 遊戲架構

此遊戲採用 **MVC (Model-View-Controller)** 架構設計：

- **Model (model.py)**：負責遊戲數據、狀態和邏輯，如網格生成、答案設置、點擊處理等。
- **View (view.py)**：負責所有 Tkinter UI 元件，包括格子按鈕、標籤和佈局。
- **Controller (controller.py)**：協調 Model 和 View，處理用戶事件和更新視圖。

這種架構確保代碼的可維護性和模組化。
